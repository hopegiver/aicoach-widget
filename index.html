🚀 사용 방법

HTML 파일에 다음 코드만 추가하세요:

<script src="https://cdn.your-domain.com/aicoach.min.js"></script>
<script>
  new AICoach({
    apiUrl: 'https://aicoach-api.your-domain.workers.dev'
  });
</script>
